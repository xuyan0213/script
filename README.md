# 青龙 PTTime每日签到

#### 介绍
PTTime 每日签到

#### 使用说明
1. 使用`ql repo https://github.com/xuyan0213/script.git "pttime"`命令拉取任务,任务执行时间默认`2 0/6 * * *`如不合适请自行修改。
2. 环境变量中增加PTTIME_COOKIE变量，cookie自行获取，不会的话百度。
3. 多个cookie可以使用使用@或者\n分割。
4. 增加PTTIME_UID变量，就是你账号的UID。
